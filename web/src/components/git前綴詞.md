# 常用前綴中文對照

| 前綴     | 中文意思        | 使用時機                                    |
| -------- | --------------- | ------------------------------------------- |
| init     | 初始化          | 專案第一次建立                              |
| feat     | 功能（feature） | 新增一個新功能                              |
| fix      | 修復            | 修掉 bugstyle樣式調整 CSS、排版，不影響邏輯 |
| refactor | 重構            | 改寫程式碼結構，但功能不變                  |
| chore    | 雜務            | 更新套件、修改設定檔等瑣事                  |
| docs     | 文件            | 新增或修改 README 等文件                    |
